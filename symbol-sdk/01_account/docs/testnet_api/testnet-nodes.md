# テストネットのノード一覧

## ノード一覧の取得元

Symbol Statistic Service のテストネットを使用する。

- `https://testnet.symbol.services/nodes`
- 例: `https://testnet.symbol.services/nodes?limit=10&ssl=true`

## 取得例（https）

サンプル取得で以下が確認できた。

- https://201-sai-dual.symboltest.net:3001
- https://2.dusanjp.com:3001
- https://sym-test-01.opening-line.jp:3001

注意: ノード一覧は随時変化するため、都度 `/nodes` を取得して確認すること。
